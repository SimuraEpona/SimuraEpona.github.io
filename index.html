问题说明

由于需要批量处理数据，并且这个数据的量很大，一次全部取出然后执行是不现实的，幸运的是 Laravel 为我们提供了 chunkById 方法来让我们方便的处理。伪代码如下

Student::query()
    ->where('is_delete', false)
    ->orderBy('id', 'DESC')
    ->chunkById(200, function($students) {
            // 在这里进行逻辑处理
    });

咋一眼看上去，并没有什么问题，但是实际执行代码的时候会发现chunkById只会执行第一次，第二次以后由于某种原因会停止执行。
