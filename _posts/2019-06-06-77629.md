> 本文翻译自 [Git Stash and Rebase in GitHub Desktop 2.0](https://laravel-news.com/git-stash-and-rebase-in-github-desktop-2-0)

近日，Github 发布了 [Github Desktop 2.0 版本](https://github.com/desktop/desktop/releases/tag/release-2.0.0)， 是对之前版本的一个重大升级。其中包含了两个最主要的功能： stash 和 rebase。

<!--more-->

我一般都是在命令行中使用这两个命令，但是我还是很喜欢看到 GitHub desktop 集成了 stash，并且还包含了 diff 功能：

![github](https://wp.laravel-news.com/wp-content/uploads/2019/06/github-desktop-2-01.png)

rebase 可能是一个比较高级的功能，所以我推荐先了解一下 rebase 的基础内容之后在进行操作。使用命令行执行 rebase 可能有些复杂，但是 GitHub desktop 让其变的非常容易操作。

有 2.0 版本的具体更新可以在[这里](https://github.blog/2019-06-05-github-desktop-expands-to-support-stashing-and-rebasing/)查看。

那么你们用的是什么软件来执行git的呢？

PS： 我一直用的都是 SourceTree。
